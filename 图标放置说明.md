# 图标放置说明

## 需要添加的图标文件

请将以下图标文件复制到 `images/` 文件夹中：

### 1. Lumirain工作室Logo
- **文件名**: `lumirain-logo.png`
- **来源**: 您提供的金色火炬logo (E6D7E196-CCB1-4007-A980-3AE3FAFA3F2B.PNG)
- **用途**: 
  - 网站导航栏logo (小尺寸)
  - 首页Hero区域的主要logo (大尺寸)
- **建议尺寸**: 200x200px 或更高分辨率

### 2. 七印应用图标
- **文件名**: `qiyin-icon.png`
- **来源**: 您提供的七印应用图标 (image.png)
- **用途**:
  - 七印页面的主要图标
  - 首页双支柱"效率"部分的图标
- **建议尺寸**: 200x200px 或更高分辨率

## 圆角效果

所有图标都已经添加了圆角效果：
- **导航栏logo**: `rounded-lg` (中等圆角)
- **页面主要图标**: `rounded-2xl` (大圆角)
- **双支柱图标**: `rounded-xl` (较大圆角)

## 图标显示位置

✅ **已添加图标的位置**:
1. 所有页面的导航栏 - Lumirain logo (32x32px)
2. 首页Hero区域 - Lumirain logo (大尺寸 96-128px)
3. 七印页面Hero区域 - 七印图标 (大尺寸 80-96px)
4. 首页双支柱"效率"部分 - 七印图标 (48x48px)

## 操作步骤

1. 将您的两个图标文件重命名为:
   - `lumirain-logo.png`
   - `qiyin-icon.png`

2. 复制到网站根目录下的 `images/` 文件夹

3. 刷新网页即可看到图标显示

## 图标要求

- **格式**: PNG (支持透明背景)
- **质量**: 高分辨率，适合web显示
- **背景**: 推荐透明背景或纯色背景
- **优化**: 压缩过的web格式，文件大小适中
