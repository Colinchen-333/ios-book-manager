# Lumirain Studio 官网

## 项目简介

这是 Lumirain Studio 的官方网站，展示了我们的产品和服务：

- **七印图书管理器** - 口袋图书馆，随扫随存
- **助残中心** - 让科技温柔突破 1 px 的门槛

## 网站结构

```
├── index.html          # 首页
├── efficiency.html     # 七印图书管理器页面
├── impact.html         # 助残中心页面  
├── about.html          # 关于我们页面
├── contact.html        # 联系我们页面
└── README.md          # 项目说明
```

## 技术栈

- **HTML5** - 结构标记
- **TailwindCSS** - 样式框架
- **Font Awesome** - 图标库
- **JavaScript** - 交互功能

## 设计特色

- 🎨 现代化响应式设计
- 🌈 渐变色彩和流畅动画
- 📱 移动端优化
- ♿ 无障碍访问支持
- 🚀 快速加载体验

## 本地运行

1. 克隆项目到本地
2. 使用任意 HTTP 服务器运行，例如：
   ```bash
   # 使用 Python
   python -m http.server 8000
   
   # 使用 Node.js
   npx serve .
   
   # 或直接在浏览器中打开 index.html
   ```
